# Documentation

## Installation
1. Unzip the package
2. Open `pages/index.html` in your browser.
3. No build steps required for the static version.
4. To customize, edit the CSS files in `assets/css/`.

## Customization
- **Colors**: Edit CSS variables in `assets/css/style.css`.
- **Fonts**: Change the Google Fonts import in HTML head.
- **Images**: Replace placeholders in `assets/images/` or src attributes.
