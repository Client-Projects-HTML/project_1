# Stackly - Telecommunications Contractor HTML Template

A professional, responsive, and accessible HTML template designed for Telecom Contractors and Industrial businesses.

## Structure
- `assets/` - CSS, JS, Images, Fonts
- `pages/` - Public facing website pages
- `dashboard/` - Admin and Client dashboards
- `documentation/` - User guides

## Features
- Bootstrap 5
- Dark Mode Support
- RTL Support
- Accessible (WCAG 2.1 AA)

## Getting Started
Open `pages/index.html` in your browser to view the home page.
